# 风味搭配与创意菜谱引擎

## 一、Foodpairing 风味科学基础

### 核心原理

Foodpairing 理论基于挥发性芳香化合物匹配：含有共同关键芳香化合物的食材，搭配在一起风味和谐。

**经典科学搭配示例**：

| 食材A | 食材B | 共享芳香化合物 | 搭配效果 |
|-------|-------|--------------|---------|
| 香蕉 | 雪梨 | 乙酸异戊酯 | 果香和谐 |
| 番茄 | 奶酪 | 4-甲基戊酸 | 经典意式 |
| 巧克力 | 咖啡 | 吡嗪类 | 深度醇厚 |
| 草莓 | 橙子 | 丁酸乙酯 | 清新果香 |
| 白巧克力 | 鱼子酱 | 三甲胺衍生物 | 咸甜碰撞 |

### 风味轮盘

**五大基础味觉**：酸、甜、苦、咸、鲜

**搭配原则**：
- 对比搭配：酸+甜（糖醋）、咸+甜（红烧）
- 协同搭配：鲜+鲜（菌菇+海鲜）、甜+甜（水果+蜂蜜）
- 抑制搭配：甜抑苦（咖啡+糖）、酸抑腻（柠檬+油炸）

## 二、君臣佐使香料理论

### 中式香料搭配体系

源自传统中医药理论，应用于烹饪香料配伍：

**猪肉**：
- 君料（主导）：八角、桂皮
- 臣料（辅助）：胡椒、干姜、甘草
- 佐使料（调和）：香叶、砂仁

**牛肉**：
- 君料：肉豆蔻、草豆蔻
- 臣料：陈皮、荜拨、甘草
- 佐使料：丁香

**羊肉**：
- 君料：草果
- 臣料：山柰、砂仁
- 佐使料：孜然、白胡椒

**鸡肉**：
- 君料：白豆蔻
- 臣料：草果、陈皮
- 佐使料：丁香、小茴香

**水产**：
- 君料：砂仁
- 臣料：芫荽子、丁香
- 佐使料：白胡椒、陈皮

### 香料使用原则

1. 君料用量最大，决定菜品基本风味方向
2. 臣料辅助君料，增强或调和主味
3. 佐使料用量最少，起画龙点睛作用
4. 总量控制：香料总量不超过食材重量的1-2%
5. 炖煮类先放，快炒类后放

## 三、烹饪技法匹配

| 食材组合 | 推荐烹饪 | 理由 | 火候要点 |
|---------|---------|------|---------|
| 嫩肉+脆蔬 | 快炒 | 保持双方口感 | 大火快炒，2-3分钟 |
| 根茎类+肉类 | 炖煮 | 软化纤维，融合风味 | 小火慢炖，1-2小时 |
| 海鲜+清淡蔬菜 | 清蒸 | 保留鲜味，避免油腻 | 中火8-12分钟 |
| 剩余米饭+杂蔬 | 炒饭/烩饭 | 重新赋予口感和风味 | 大火翻炒，米粒分明 |
| 豆腐+菌菇 | 红烧/煮汤 | 吸收汤汁，提升鲜味 | 中小火，入味为主 |
| 鸡蛋+任意蔬菜 | 炒/蒸/煮 | 万能搭配 | 中火，蛋嫩即出 |
| 面条+剩菜 | 拌面/汤面 | 一锅解决 | 面条另煮，菜做浇头 |
| 土豆+肉类 | 炖/烤 | 土豆吸肉汁 | 小火慢炖至土豆软糯 |

## 四、质地互补原则

| 质地类型 | 代表食材 | 最佳互补 |
|---------|---------|---------|
| 脆嫩 | 黄瓜、木耳、豆芽 | 软糯（土豆、豆腐） |
| 软糯 | 土豆、豆腐、南瓜 | 脆嫩（黄瓜、木耳） |
| 爽滑 | 鱼片、蛋羹 | 粗糙（豆芽、芹菜） |
| 粗糙 | 芹菜、豆芽、笋 | 爽滑（鱼片、蛋羹） |
| 酥脆 | 油条、锅巴、炸物 | 柔软（豆腐、面条） |
| Q弹 | 虾滑、鱼丸、年糕 | 松软（米饭、馒头） |

## 五、色彩对比（五色原则）

中式烹饪讲究"色香味俱全"，五色搭配提升食欲：

| 颜色 | 代表食材 | 营养侧重 |
|------|---------|---------|
| 红 | 番茄、红椒、肉类、红枣 | 番茄红素、铁 |
| 绿 | 青菜、葱、西兰花、青椒 | 维生素C、叶绿素 |
| 黄 | 鸡蛋、姜、玉米、南瓜 | 胡萝卜素、蛋白质 |
| 白 | 豆腐、蒜、白萝卜、山药 | 蛋白质、膳食纤维 |
| 黑 | 木耳、香菇、紫菜、豆豉 | 铁、多糖 |

**搭配建议**：一道菜至少覆盖3种颜色，视觉丰富度与营养均衡度正相关。

## 六、食材替代方案

当用户缺少某种食材时，提供替代建议：

| 缺少食材 | 替代方案 | 风味差异 |
|---------|---------|---------|
| 蚝油 | 酱油+白糖+少许料酒 | 咸鲜为主，缺少蚝香 |
| 料酒 | 黄酒/白酒（少量）/姜汁 | 去腥效果相近 |
| 淀粉 | 面粉（用量减半）/玉米粉 | 口感略有差异 |
| 鸡精 | 浓汤宝/菌菇粉 | 鲜味来源不同 |
| 醋 | 柠檬汁/番茄 | 酸度不同，用量需调整 |
| 生抽 | 老抽（减量）+盐 | 颜色更深，咸度更高 |
| 老抽 | 生抽+少许白糖 | 上色效果差，需加糖调色 |
| 八角 | 桂皮+少许五香粉 | 风味接近但不完全相同 |
| 鲜辣椒 | 干辣椒/辣椒酱/辣椒油 | 辣度不同，用量需调整 |
| 鲜姜 | 姜粉（1/3用量） | 风味略弱 |
| 鲜蒜 | 蒜粉（1/3用量）/蒜蓉酱 | 风味略弱 |
| 葱 | 韭菜/洋葱（少量） | 风味不同但可替代 |
| 牛奶 | 豆浆/椰奶 | 口感和风味差异较大 |
| 黄油 | 植物油 | 缺少奶香，口感略差 |
| 面包糠 | 压碎的饼干/馒头渣 | 口感接近 |

## 七、季节时令适配

### 春季（3-5月）
- 时令食材：春笋、香椿、荠菜、豌豆、蚕豆
- 推荐搭配：春笋+腊肉、香椿+鸡蛋、荠菜+豆腐
- 烹饪偏好：清淡、鲜嫩

### 夏季（6-8月）
- 时令食材：黄瓜、番茄、丝瓜、苦瓜、毛豆
- 推荐搭配：苦瓜+鸡蛋、丝瓜+蛤蜊、毛豆+榨菜
- 烹饪偏好：爽口、开胃、少油

### 秋季（9-11月）
- 时令食材：南瓜、板栗、山药、莲藕、柿子
- 推荐搭配：板栗+鸡块、山药+排骨、莲藕+猪蹄
- 烹饪偏好：温补、醇厚

### 冬季（12-2月）
- 时令食材：白萝卜、大白菜、羊肉、红薯、荸荠
- 推荐搭配：萝卜+羊肉、白菜+豆腐、红薯+姜
- 烹饪偏好：暖身、炖煮

## 八、一人食小份量调整

| 标准菜谱用量 | 一人食用量 | 备注 |
|------------|-----------|------|
| 主料 300g | 100-150g | 一人份蛋白质足够 |
| 蔬菜 200g | 80-120g | 保证膳食纤维 |
| 米饭 2碗 | 1碗（约150g熟饭） | 生米约50g |
| 面条 200g | 80-100g | 干面重量 |
| 油 2汤匙 | 1汤匙（约10ml） | 控油 |
| 盐 1茶匙 | 1/2茶匙（约2-3g） | 控盐 |
| 酱油 2汤匙 | 1汤匙 | 适量调整 |

**一人食技巧**：
- 一锅出：炒菜+主食合一（炒饭、拌面、焖饭）
- 分装冷冻：多做几份，冷冻保存
- 万能酱汁：提前调好酱汁，随用随取
- 小锅具：使用小煎锅/小奶锅，省时省力

## 九、经典风味搭配速查

### 万能搭配组合

| 组合 | 适用场景 | 调味要点 |
|------|---------|---------|
| 番茄+鸡蛋 | 炒菜、汤、面 | 番茄先炒出汁 |
| 土豆+牛肉 | 炖煮、咖喱 | 土豆后放防碎 |
| 豆腐+菌菇 | 红烧、煮汤 | 豆腐先煎再烧 |
| 鸡蛋+韭菜 | 炒菜、饺子 | 大火快炒 |
| 白菜+豆腐 | 炖煮、火锅 | 白菜帮先下锅 |
| 木耳+山药 | 炒菜 | 山药先焯水 |
| 青椒+肉丝 | 炒菜 | 肉丝先滑炒 |
| 茄子+蒜 | 红烧、凉拌 | 茄子先蒸/微波减少吸油 |

### 创意碰撞组合

| 组合 | 惊喜点 | 操作要点 |
|------|--------|---------|
| 西瓜皮+蒜 | 清脆爽口 | 去外皮切薄片，盐腌去水 |
| 剩饭+芝士 | 焗饭/锅巴 | 饭铺底，芝士覆盖烤制 |
| 香蕉+鸡蛋 | 香蕉煎饼 | 香蕉压泥混合蛋液 |
| 土豆+芝士 | 芝士焗土豆 | 土豆微波8分钟，挖空填料 |
| 馒头+鸡蛋 | 馒头披萨 | 馒头切片裹蛋液煎 |
| 剩面条+鸡蛋 | 面条饼 | 面条+蛋液混合煎制 |
| 豆腐+皮蛋 | 凉拌豆腐 | 皮蛋切碎拌入 |
