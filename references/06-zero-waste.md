## 三、冰箱整理月度计划

### 第1周：检查与清理

1. 逐层检查所有食材，清除过期/变质项
2. 擦拭内壁和隔板（白醋+水1:1）
3. 检查密封条是否老化
4. 调整温度：冷藏2-4°C，冷冻≤-18°C

### 第2周：分区整理

1. 按分区标准重新摆放（上层熟食、下层生食）
2. 所有食材贴标签（名称+日期+期限）
3. 临期食材移至前部/顶部
4. 气味重食材密封盒存放

### 第3周：消耗临期

1. 优先使用🟡级食材
2. 临期蔬菜焯水冷冻
3. 临期水果做果酱/冷冻
4. 临期肉类分装冷冻

### 第4周：深度清洁

1. 断电→清空所有食材
2. 可拆卸部件温水+小苏打清洗
3. 内壁白醋擦拭→晾干
4. 密封条温水擦拭
5. 复位后等待30分钟再放回食材

## 四、冰箱存放常见误区

| 误区 | 正确做法 | 原因 |
|------|---------|------|
| 热菜放凉再入冰箱 | 温热即可放入（60°C以下） | 常温放凉过程在危险温度区停留过久 |
| 所有蔬菜都放冰箱 | 土豆/洋葱/大蒜/香蕉常温存放 | 土豆冷藏淀粉转化变甜，洋葱大蒜易受潮发芽 |
| 塑料袋直接装肉 | 密封盒/密封袋分装 | 塑料袋不密封，交叉污染和串味风险 |
| 反复解冻冷冻 | 分装单次用量，一次解冻 | 反复解冻品质下降，细菌繁殖 |
| 冰箱塞满 | 保留30%空间 | 冷气需要循环空间，塞满影响制冷效果 |
| 鸡蛋放门架 | 放中层 | 门架温度波动大，鸡蛋敏感 |
| 番茄放冰箱 | 未熟常温，熟后短期冷藏 | 冷藏影响番茄风味和质地 |
| 剩菜直接放冰箱 | 2小时内密封冷藏 | 常温超2小时进入危险温度区 |
