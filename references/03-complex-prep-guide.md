# 复杂食材精细处理指南

## 一、海鲜类处理SOP

### 1. 鱼类（以鲈鱼/草鱼/鲫鱼为例）

**处理难点**：去鳞、去鳃、去内脏、去腥

**标准SOP**：

1. **去鳞**：逆鳞方向从尾部向头部刮净，腹部细鳞重点处理
2. **去鳃**：剪开鳃盖，拽出完整拽出完整鳃丝（鳃是腥味集中区）
3. **去内脏**：腹部剖开，取出内脏团，注意保留鱼籽/鱼泡（如需要）
4. **刮净黑膜**：腹腔内黑色薄膜是土腥味主要来源，必须刮净
5. **冲洗**：流水冲洗腹腔至无血水
6. **腌制去腥**：葱姜盐水（葱2根+姜3片+盐5g+水200ml）浸泡10分钟，或料酒+柠檬汁涂抹鱼身

**安全要点**：
- 鱼鳃必须完全去除，是腥味和细菌集中区
- 腹腔黑膜必须刮净
- 必须彻底煮熟，中心温度达70°C以上

**科学去腥原理**：
- 鱼腥味来自三甲胺(TMA)和氧化脂肪
- 酸性食材（柠檬/醋/番茄）中和碱性三甲胺
- 酒精萃取挥发性腥味分子
- 葱属植物（葱/姜/蒜）含硫化物抑制腥味产生

### 2. 鱿鱼/章鱼

**处理难点**：去皮、去内脏、去腥、嫩化

**标准SOP**：

1. **拉出头**：抓住头部向外拉，带出内脏团
2. **取出软骨**：在鱿鱼筒内取出透明塑料状软骨
3. **撕去紫膜**：剥去表面紫色薄膜（腥味来源）
4. **剪开鱿鱼筒**：纵向剪开，打十字花刀（深度2/3，不切断）
5. **处理鱿鱼须**：去除眼睛和硬嘴（牙齿状结构）
6. **嫩化处理**：用萝卜汁腌制或反复摔打（破坏肌肉纤维）

**安全要点**：
- 内脏可能有墨汁，注意清洁
- 眼睛和硬嘴必须去除
- 焯水时间不宜过长（30秒-1分钟），否则变硬

### 3. 贝类（蛤蜊/蛏子/扇贝）

**处理难点**：吐沙、去内脏

**标准SOP**：

1. **盐水吐沙**：清水+盐（3%浓度，模拟海水）+几滴香油，浸泡2-4小时，中途换水2-3次
2. **刷洗外壳**：硬毛刷清洁表面泥沙
3. **去内脏黑线**：焯水至开口后，撕掉外圈黑色内脏线
4. **冲洗**：流水冲净残留泥沙

**安全要点**：
- 未开口的贝类=已死，不可食用，必须丢弃
- 香油隔绝水面空气，迫使贝类开口吐沙
- 盐水浓度3%模拟海水环境，促进其生理活动

### 4. 虾类

**处理难点**：去虾线、去头

**标准SOP**：

1. **去虾线**：背部第二节用牙签挑出黑色肠线（虾线=消化道，含泥沙和细菌）
2. **剪须脚**：剪去长须和细脚（避免扎嘴，提升美观）
3. **去头（可选）**：扭下头部，挤出胃囊（头部重金属富集区）

**安全要点**：
- 虾线建议去除，但少量残留不会中毒
- 虾头发黑不一定是变质（可能是酪氨酸酶氧化），但头部重金属含量较高
- 必须彻底煮熟

### 5. 螃蟹

**处理难点**：绑扎、清洗、去鳃

**标准SOP**：

1. **刷洗外壳**：流水下用刷子清洁蟹壳和蟹腿
2. **掀开蟹壳**：从尾部掀开，去除蟹鳃（两侧灰白色羽毛状组织）
3. **去内脏**：去除蟹心（六角形白色小块，寒性大）、蟹胃（三角锥形）
4. **切块**：对半切开或四等分

**安全要点**：
- 死蟹绝对不可食用（组胺中毒风险）
- 蟹鳃、蟹心、蟹胃必须去除
- 必须彻底煮熟

## 二、内脏类处理SOP

### 1. 猪肝

**处理难点**：去血水、去腥

**标准SOP**：

1. **切片**：逆纹理切2mm薄片（越薄越易熟透，口感越嫩）
2. **面粉搓洗**：加2勺面粉抓揉2分钟，吸附血水和毒素
3. **清水漂洗**：流水冲洗至水清（至少3次）
4. **料酒腌制**：料酒10ml+姜片+淀粉5g，腌制15分钟

**安全要点**：
- 必须彻底煮熟，中心温度达70°C以上
- 面粉搓洗是关键步骤，不可省略
- 猪肝维生素A含量高，不宜频繁大量食用

### 2. 猪肚

**处理难点**：去腥、去黏液、去白膜

**标准SOP**：

1. **翻面**：将猪肚完全翻至内壁朝外
2. **盐醋搓洗**：盐50g+白醋30ml，反复揉搓5分钟（去除黏液和异味）
3. **焯水**：冷水下锅，加葱姜料酒，煮沸后煮3分钟
4. **刮净白膜**：用刀刮去内壁白色脂肪膜（腥味主要来源）
5. **二次清洗**：流水冲净

**安全要点**：
- 白膜必须去除，是腥味核心
- 焯水必须冷水下锅，利于异味排出
- 需长时间炖煮（高压锅30分钟/普通锅1.5小时）确保熟透

### 3. 猪大肠

**处理难点**：去腥、去黏液、去异味

**标准SOP**：

1. **翻面**：将肠管完全翻转
2. **面粉/可乐搓洗**：面粉3勺或可乐200ml，抓揉5分钟（面粉吸附黏液，可乐的碳酸和磷酸分解有机物）
3. **焯水**：冷水下锅，加料酒姜片，煮沸去沫
4. **二次清洗**：翻面还原，再次冲洗

**安全要点**：
- 必须彻底煮熟
- 可乐搓洗是家庭最有效的去异味方法之一
- 内壁油脂可视个人喜好保留或去除

### 4. 猪腰

**处理难点**：去腰臊（腥骚味核心）

**标准SOP**：

1. **对半切开**：平刀剖开成两半
2. **剔除腰臊**：用刀片去中央白色筋膜（腰臊=腥味和骚味核心，必须完全去除）
3. **斜刀切片**：45度斜切，片薄易熟
4. **料酒腌制**：料酒+胡椒粉+淀粉，腌制10分钟

**安全要点**：
- 白色腰臊必须完全剔除，残留即有骚味
- 切片后可再过一遍清水漂洗
- 快炒最佳，过老则硬

### 5. 鸡鸭内脏

**处理难点**：去腥、处理苦胆

**标准SOP**：

1. **盐搓洗**：盐搓洗去黏液
2. **去苦胆**：小心摘除胆囊（弄破则整只丢弃，胆汁极苦且有毒性）
3. **料酒姜片腌制**：腌制15分钟
4. **焯水**：冷水下锅，加料酒姜片

**安全要点**：
- 苦胆弄破后胆汁污染的肉不可食用
- 鸡胗需去内壁黄膜
- 必须彻底煮熟

## 三、蔬菜类处理SOP

### 1. 花菜/西兰花

**处理难点**：藏虫、农药残留

**标准SOP**：

1. **拆分小朵**：切分成一口大小，保留部分茎部
2. **盐水浸泡**：淡盐水（1%浓度）浸泡15分钟（驱虫杀菌）
3. **冲洗**：流水冲净盐分和小虫

**安全要点**：
- 盐水浸泡是驱虫最有效方法
- 茎部去皮后可食用，不要浪费

### 2. 菌菇类

**处理难点**：吸水影响风味

**标准SOP**：

1. **干刷去土**：用软毛刷清洁表面（忌水洗，菌菇吸水后风味流失）
2. **快速冲洗**：如需冲洗，流水快速冲过，立即挤干水分
3. **切片/撕块**：根据菜品需求处理

**安全要点**：
- 忌长时间浸泡，吸水后口感变差
- 野生菌一律不处理，仅推荐人工培育品种

### 3. 竹笋

**处理难点**：去涩、去草酸

**标准SOP**：

1. **剥壳**：逐层剥去硬壳，切去老根
2. **焯水**：冷水下锅，煮沸后继续煮5-10分钟（去除草酸和涩味）
3. **冷水浸泡**：换冷水浸泡2小时（进一步去涩）

**安全要点**：
- 焯水必须冷水下锅
- 苦笋涩味更重，需延长焯水时间

### 4. 菠菜/苋菜等高草酸蔬菜

**处理难点**：去草酸

**标准SOP**：

1. **焯水**：沸水焯烫30秒-1分钟（去除大部分草酸）
2. **过冷水**：保持翠绿色泽

**安全要点**：
- 草酸影响钙吸收，焯水可去除60-70%
- 焯水时间不宜过长，营养流失

### 5. 四季豆/扁豆

**处理难点**：去毒素

**标准SOP**：

1. **去筋**：撕去两侧老筋
2. **焯水**：沸水焯烫3-5分钟
3. **充分炒熟**：中心温度须达100°C持续10分钟以上

**安全要点**：
- 含皂素和植物血细胞凝集素，必须彻底煮熟
- 颜色由翠绿变为暗绿、口感无生豆腥味即为熟透
- 严禁生食/半生食

## 四、干货类处理SOP

### 1. 木耳/银耳

**处理难点**：泡发安全

**标准SOP**：

1. **冷水泡发**：冷水浸泡1-2小时即用
2. **清洗**：流水冲洗，去除根部杂质
3. **焯水**：沸水焯烫1-2分钟

**安全要点**：
- ⚠️ 禁止常温过夜泡发！易滋生椰毒假单胞菌，产生米酵菌酸毒素
- 米酵菌酸毒素耐高温，常规烹饪无法分解
- 致死率50%以上
- 泡发超过4小时需冷藏进行
- 泡发后黏滑、异味立即丢弃

### 2. 海带

**处理难点**：去盐、泡发

**标准SOP**：

1. **干刷去盐**：表面盐粒刷去
2. **温水泡发**：温水浸泡20分钟
3. **冲洗**：流水冲净

### 3. 粉条/腐竹

**处理难点**：均匀泡发

**标准SOP**：

1. **温水泡发**：温水浸泡30分钟
2. **忌热水**：热水泡发外烂内硬
3. **剪段**：泡发后剪成适口长度

### 4. 干香菇

**标准SOP**：

1. **温水泡发**：温水浸泡30分钟-1小时
2. **泡发水保留**：沉淀后上层清液可做汤底（含鲜味物质）
3. **清洗**：流水冲洗伞褶内杂质

## 五、黏液类食材处理

### 1. 山药

**处理难点**：黏液致痒

**标准SOP**：

1. **戴手套**：防止黏液接触皮肤致痒
2. **削皮**：戴手套削去外皮
3. **醋水浸泡**：切后入醋水防氧化变色
4. **如已手痒**：白醋涂抹或火上烤手（分解皂角素）

### 2. 秋葵

**标准SOP**：

1. **整根焯水**：沸水焯烫30秒（保留黏液营养）
2. **去头尾**：切去两端
3. **忌切后焯水**：黏液流失

### 3. 莲藕

**标准SOP**：

1. **去皮切片**：去皮后立即入醋水（防氧化变黑）
2. **焯水**：沸水焯烫1-2分钟
3. **过冷水**：保持脆爽口感
