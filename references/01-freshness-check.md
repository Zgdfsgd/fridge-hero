# 食材鉴鲜与安全判定标准

## 一、四维鉴鲜体系

### 1. 外观检查

| 检查项 | 正常状态 | 异常信号 |
|--------|---------|---------|
| 表面光泽 | 自然光泽、湿润度适中 | 暗淡无光、异常反光 |
| 完整性 | 无破损、无异物 | 破损、霉斑、白毛、黑点 |
| 干湿度 | 适度湿润或干燥 | 渗水、干裂、脱水皱缩 |
| 均匀性 | 颜色质地均匀 | 局部变色、局部软化 |

### 2. 颜色检查

| 食材类别 | 正常颜色 | 变质颜色 |
|---------|---------|---------|
| 红肉 | 鲜红/暗红（牛肉偏深） | 灰褐、灰绿、发黑 |
| 禽肉 | 粉白/淡粉 | 灰绿、发黄、暗灰 |
| 鱼肉 | 白色/淡粉、半透明 | 发黄、发灰、浑浊 |
| 虾 | 青灰/青白（生）/粉红（熟） | 黑斑、发黑、暗灰 |
| 绿叶蔬菜 | 翠绿/深绿 | 发黄、发黑、褐斑 |
| 根茎类 | 均匀本色 | 发绿（土豆）、黑斑、霉点 |
| 菌菇 | 本色均匀 | 发黑、发黄、白毛 |
| 豆制品 | 白色/淡黄 | 发红、发灰、暗沉 |
| 鸡蛋 | 蛋白透明、蛋黄完整 | 蛋白浑浊、蛋黄散开、黑斑 |

### 3. 气味检查

| 气味类型 | 可能原因 | 判定 |
|---------|---------|------|
| 酸味/酸败味 | 细菌分解糖类/脂肪氧化 | ❌变质 |
| 腐臭味 | 蛋白质分解产生胺类 | ❌变质 |
| 哈喇味 | 脂肪氧化酸败 | ❌变质 |
| 氨味 | 蛋白质严重分解 | ❌变质 |
| 酒精味/发酵味 | 酵母菌繁殖 | ⚠️视程度判定 |
| 泥土味/腥味（正常范围） | 食材本身气味 | ✅正常 |
| 无味/气味减弱 | 可能失水或存放较久 | ⚠️结合其他维度判定 |

### 4. 触感检查

| 食材类别 | 正常触感 | 异常触感 |
|---------|---------|---------|
| 肉类 | 弹性良好，指压即恢复 | 黏滑、指压痕不恢复、松散 |
| 鱼类 | 肉质紧实有弹性 | 松散、易碎、糊化 |
| 蔬菜 | 脆嫩、挺拔 | 软烂、渗水、黏滑 |
| 菌菇 | 干爽、紧实 | 黏液、软化 |
| 豆制品 | 紧实、有弹性 | 发黏、出水、松散 |
| 鸡蛋 | 摇晃无声 | 摇晃有水声（散黄） |

## 三、感官自检三步法

当用户对食材状态不确定时，引导执行：

1. **看**：仔细观察表面有无霉斑、变色、黏液、虫蛀
2. **闻**：靠近闻气味，有无酸腐、氨味、哈喇味
3. **摸**：轻按表面，有无黏滑、软烂、弹性丧失

**核心原则：当不确定时，宁丢勿烹。**

## 四、危险温度区规则

依据 FDA 食品法典和 WHO 食品安全五要点：

- **危险温度区**：5°C - 60°C
- **2小时规则**：食材在危险温度区停留超过2小时，即存在高风险
- **1小时规则**：气温超过32°C时，缩短为1小时
- **安全加热标准**：中心温度须达70°C以上
- **安全冷藏标准**：0-4°C（依据 GB 31605-2020）
- **安全冷冻标准**：≤-18°C（依据 GB 31605-2020）

## 五、特殊判定场景

### "切掉坏的部分，剩下的能吃吗？"

| 食材类型 | 回答 | 科学依据 |
|---------|------|---------|
| 霉变面包/水果 | ❌整块丢弃 | 霉菌菌丝肉眼不可见部分已深入 |
| 霉变坚果/粮食 | ❌整块丢弃 | 黄曲霉毒素耐热280°C，扩散范围大 |
| 腐烂生姜 | ❌整块丢弃 | 黄樟素已扩散至看似完好的部分 |
| 发芽土豆 | ⚠️轻微发芽可剜除芽眼及周围大面积组织；严重发芽直接丢弃 | 龙葵素耐高温，已扩散至周围组织 |
| 碰伤的水果 | ✅切除碰伤部分即可 | 机械损伤非微生物污染 |
| 轻微蔫软的蔬菜 | ✅可食用，焯水或做汤 | 仅失水，无变质 |

### 冷冻食品变质判定

| 特征 | 含义 | 判定 |
|------|------|------|
| 大量冰霜/冰晶 | 经历温度波动或存放过久 | ⚠️品质下降，但仍安全 |
| 严重冻烧（灰白干燥斑） | 表面脱水氧化 | ⚠️口感差，切除后可食用 |
| 包装内大量液体（解冻后） | 细胞破裂严重 | ⚠️品质差，尽快使用 |
| 异味/变色 | 可能已变质 | 🔴丢弃 |
