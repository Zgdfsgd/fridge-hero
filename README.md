# 🧊 冰箱清道夫 — Fridge Hero

**合法 · 科学 · 实用的冰箱食材AI管家**

## ✨ 核心能力

| 模式 | 名称 | 核心能力 | 创新点 |
|------|------|----------|--------|
| 🔍 一 | 食材鉴鲜与安全判定 | 四维鉴鲜（外观/颜色/气味/触感）+ 三级警示（🔴🟡🟢） | GB 31605-2020温度标准 + 危险温度区2小时规则 |
| 🧊 二 | 科学储存与时效管理 | 冷藏/冷冻/常温时限 + FIFO原则 + 冰箱分区 | GB 7718-2025到期日新规 + 临期拯救方案 |
| 🔪 三 | 复杂食材精细处理 | 海鲜去鳞去腥/内脏清洗/蔬菜去涩/干货泡发 全流程SOP | 科学去腥原理（三甲胺中和）+ 家庭妙招 |
| 🍳 四 | 创意菜谱智能生成 | 风味搭配 + 君臣佐使 + 难度分级 + 一人食适配 + 食材替代 | Foodpairing芳香化合物匹配 + 季节时令适配 |
| ⚠️ 五 | 高危食材专项警示 | 发芽土豆/四季豆/鲜黄花菜/野生菌/久泡干货 强制预警 | 中毒原理科普 + 强制丢弃标准 + 误食急救指引 |
| ♻️ 六 | 零浪费与冰箱优化 | 边角料再利用 + 厨余变身 + 冰箱整理方案 + 食材相克辟谣 | 科学辟谣（中国营养学会背书）+ 碳足迹意识 |

## 🌟 创新亮点

- 🔬 **科学去腥原理** — 不只教"怎么做"，还讲"为什么"：三甲胺(TMA)中和原理
- 🧪 **Foodpairing芳香化合物匹配** — 基于科学的风味搭配，非凭感觉
- 🚫 **食材相克科学辟谣** — 中国营养学会实验数据辟谣，区分谣言与真实风险
- 📊 **GB 31605-2020 + GB 7718-2025 双国标支撑** — 冷链卫生规范 + 新版标签通则
- ♻️ **零浪费体系** — 边角料变身 + 临期拯救 + 碳足迹意识
- 👤 **一人食适配** — 所有菜谱支持一人份量调整
